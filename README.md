
Config stuff for i3, neovim, and some other stuff

# Installation

## Arch Linux

### Please install yay before continuing.

Picom Rounded Corners
```
yay -S picom-ibhagwan-git
```

Changing Brightness
```
sudo pacman -S xrandr
sudo pacman -S brightnessctl (just in case)
```
